The **BUBT Canteen Food Ordering System (CFOS)** is a software application developed to simplify and automate the food ordering process within the university canteen. It allows students and staff to browse available food items, add them to a cart, and place orders efficiently, reducing long queues and manual errors. The system also enables canteen staff to manage orders, update menu items, and track daily transactions. Developed using the C programming language, this project aims to improve service speed, accuracy, and overall user experience in the university canteen environment.
